# NEET Empowerment App

This repository provides a simple web application written in PHP and JavaScript. It demonstrates a login page, a menu with various cards, a group chat, and a form to submit ideas for funding.

## Running locally

You need PHP installed. Launch the built-in web server inside the `public` directory:

```bash
php -S localhost:8000 -t public
```

Then open [http://localhost:8000/index.php](http://localhost:8000/index.php) in your browser.
